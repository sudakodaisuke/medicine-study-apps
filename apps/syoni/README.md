# 小児+先発後発 カメラ改造版

小児用量と先発・後発医薬品の学習を、カード学習やクイズで復習するための静的Webアプリです。カメラ機能を改造した版で、手のひら相棒カメラなどの学習補助UIを含みます。

## Files

- `index.html` - main app
- `manifest.webmanifest` - PWA manifest
- `sw.js` - service worker
- `data.csv` - study data
- `assets/` - icons, audio, buddy, and game assets

## Run Locally

Static hostingでそのまま配信できます。ローカル確認は任意の静的サーバーでルートディレクトリを開いてください。
